NHẠC NỀN / BACKGROUND MUSIC
===========================

Đặt file nhạc của bạn tại đây với tên:  background.mp3
Place your music file here, named:      background.mp3

-> Trang web sẽ tự động phát file này khi khách mở thiệp.
-> The site will play this file automatically when the guest opens the invitation.

Nếu chưa có file, site sẽ tạm dùng một bản nhạc mẫu online
(xem CONFIG.music.fallback trong assets/js/config.js).
If this file is missing, the site falls back to a sample track online
(see CONFIG.music.fallback in assets/js/config.js).

Gợi ý nhạc cưới không bản quyền: Pixabay Music, YouTube Audio Library, Bensound.
